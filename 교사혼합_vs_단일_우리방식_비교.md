# "교사 섞기 vs 단일" 의견 — 우리 방식과 비교

> 일자: 2026-07-03 · 질문: 공유된 의견(교사 대조·합의필터링 방식)과 **우리 실제 학습이 비슷한가?**
> 답 요약: **핵심 철학은 같고(로짓 평균 X, LLM이 라벨링, 단일 일관 교사, 보수적 안전), 일부는 우리가 더 나아갔으며(soft-label·균형증강·앙상블 floor), 몇 가지 좋은 아이디어는 아직 안 함(2교사 합의필터·능동학습·사람 골드셋).**

---

## 0. 먼저: 교사 성능 실측 (의견의 "채점자부터 재라" 조언 = 이미 함)

| 교사 | macroF1 | C재현율 |
|---|---|---|
| **o4-mini** | **0.972** | 1.0 |
| gpt-4o | 0.915 | 1.0 |
| gpt-5.4 | 0.904 | 1.0 |
| gemma2:9b | 0.873 | 0.917 |

→ **최강 교사 = o4-mini**(GPT-5.4·gpt-4o보다 높음). 우리 주력 교사셋이 이미 o4-mini라 "제일 잘하는 한 명" 선택은 이미 최적.

---

## 1. 의견과 우리 방식 대조표

| 의견의 권고 | 우리 현황 | 판정 |
|---|---|---|
| 로짓 평균 앙상블 증류는 토크나이저 달라 불가 → **LLM이 문서를 라벨링** | 정확히 그렇게 함(교사가 O/S/C **확률분포** 부여 → KD) | ✅ 동일(오히려 hard가 아닌 **soft-label**로 더 나아감) |
| **단일 일관 교사**(작은 학생엔 혼합 라벨이 독) | 세트별 단일 교사(o4-mini 주력) | ✅ 동일 |
| 보수적 안전(의심→높은 등급) | 구현하되 **위치가 다름**: 교사합의가 아니라 **추론 규칙 floor + C재현율 우선 best-save**로 | ◑ 목표 동일, 수단 다름 |
| 다중 교사 **합의 필터**(주교사+검산, 불일치→사람) | **미구현**(현재 단일 교사, 검산 루프 없음) | ✗ 미적용 |
| 교사 불일치 = **어려운 문서 탐지(능동학습)** | 미구현(대신 난이도·적대 문서로 경계 커버) | ✗ 미적용 |
| **사람 골드셋 200~500**으로 교사 정확도 검증 | 미보유(held-out은 LLM생성·정책라벨) | ✗ 미적용(갭) |
| **문서 생성은 다양하게**(여러 모델/템플릿) | 포맷·난이도·적대·PII주입 다양성 O, 그러나 **LLM 문체 다양성은 제한**(주로 gpt-4o 생성) | ◑ 부분 적용 |
| OOD 붕괴(1.00→0.56)는 생성 균질 탓 | **원인 진단·정직 held-out으로 대응**. 현재 base 전체 파이프라인 **0.873**(0.56 아님) | ✅ 진단 일치, 이미 크게 개선 |

---

## 2. 같은 점 (핵심 철학 일치)

- **"로짓 평균 말고 LLM이 라벨링"** — 의견의 핵심 교정이 곧 우리 방식. 게다가 우리는 hard 라벨이 아니라
  **확률분포(soft-label)** 를 전이 → 경계 뉘앙스까지 학습(의견보다 정보량↑).
- **단일 일관 교사** — 작은 학생(small/base)엔 일관 정답지가 유리하다는 논지, 우리도 세트별 단일 교사.
- **보수적 안전** — "의심→높은 등급" 철학 동일. 우리는 이를 **추론단(규칙 floor·C재현율 우선)** 에서 강제 → 전 모델 유출 0.
- **OOD 붕괴 경계** — 의견이 지적한 1.00→0.56을 우리가 실제 겪고 진단, **정직 held-out**으로 상시 감시.

## 3. 우리가 더 한 것 (의견에 없는 강화)

- **soft-label KD**(확률분포 모방) — 단순 합의 hard-label보다 풍부.
- **클래스 균형 + C경계 증강** — S 기아·C/S 경계 붕괴를 데이터로 교정(의견 미언급).
- **앙상블 튜닝(soft·뉴럴가중) + 규칙 floor** — 뉴럴 실력을 안전하게 최종 등급에 반영.
- **다중 백본 라인업**(small/base/large) 3축 선택 — 용도별 배포.
- **교사 성능 실측 비교**(o4-mini·gpt-4o·gpt-5.4·gemma) — 의견의 "먼저 재라"를 수행.

## 4. 아직 안 한 것 (의견에서 채택할 좋은 아이디어)

1. **2교사 합의 필터 + 능동학습** — o4-mini(주) + gpt-4o(검산). 두 교사 **불일치 문서 = 경계/어려운 문서**로
   골라 가중 학습/사람 검토 → 같은 예산 대비 최대 향상. **가성비 높은 다음 개선.**
2. **사람 골드셋(200~500)** — LLM 라벨이 아닌 사람 확정 셋으로 교사·학생 정확도의 **절대 기준** 확보
   (상업화 SLA·검증에 필수). 현재 갭.
3. **문서 생성 다양성 확대** — o4-mini/gemma/실제 HWPX 템플릿까지 섞어 문체 균질성↓(OOD 추가 방어).

---

## 5. 결론 (한 줄)

> **핵심은 이미 이 의견대로 하고 있다**(로짓 평균 X·LLM 라벨링·단일 최강 교사=o4-mini·보수적 안전·OOD 감시),
> **일부는 더 앞서 있고**(soft-label·균형증강·앙상블 floor·라인업), **2교사 합의필터+능동학습·사람 골드셋·생성 다양성**은
> 아직 안 한 좋은 아이디어라 **다음 개선으로 채택 권장**.

> 참고: 의견이 전제한 "1.00→0.56"은 초기 상태였고, 현재는 base 전체 파이프라인 **0.873**·라지 **0.917**(전 모델 유출0)로 개선됨.
