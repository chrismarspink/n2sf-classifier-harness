안전 우선 — 기밀은 하나도 놓치지 않는다

전 모델·전 언어(한국어·영어·중국어·일본어)에서 기밀 재현율 1.00, 과소분류(유출) 0. 뉴럴이 놓칠 수 있는 경계 문서는 규칙 floor(주민번호·카드·기밀표지 등 언어무관 강식별자)가 이중으로 차단합니다. 과대분류(공개→민감)는 사람이 재검토하면 되지만, 유출은 원천 차단이 원칙입니다.

어떻게 만들었나

  • 클라우드 LLM(교사)이 문서에 O/S/C 확률분포를 부여 → 로컬 모델이 모방(soft-label 지식증류)
  • 교사엔 합성 문서만 전송(실기밀 미사용) · 추론엔 LLM 호출 0
  • 클래스 균형 + C경계·다국어 증강으로 누락 0 유지하며 정확도↑
  • 3-tier(정규식 → NER → 증류 뉴럴) + 튜닝 앙상블(뉴럴 가중)
  • 설명가능성: 어떤 검출이 등급을 만들었는지 가법적 기여도(SHAP) 제시

배포 · 온프레미스

  • Hugging Face(공개/사설) — from_pretrained()로 어디서나 (docker pull 개념)
  • Docker 이미지 — 모델+엔진+API, docker run 한 줄
  • 에어갭 — 모델 tar / 오프라인 이미지 반입, 외부 접근 0
  • GPU 불필요 — 일반 서버·PC CPU로 동작
  • 고객 정책(등급체계·규정) GradeProfile로 주입
진행 중 · 다음 갱신 — 소형 다국어 모델(multilingual-e5-small / MiniLM-L12)을 다국어 교사 라벨로 증류 중입니다. 완료 시 "다국어 경량" 티어를 매트릭스에 추가합니다. (현재 small은 한국어 전용, 다국어는 base/large가 담당.)
