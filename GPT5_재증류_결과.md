# GPT-5.4 교사 재증류 결과 (item1) — base 0.85+ 안전 달성

> 일자: 2026-07-03 · 목적: GPT-5.4를 교사로 **S+C경계 균형 증강 + 재증류** → base를 누락0 유지하며 0.8+로.
> 결과: base(mDeBERTa) **전체 파이프라인 0.873 · C재현율 1.00(유출0)**. 0.8·0.85 안전하게 초과. **승격 완료.**

---

## 1. 무엇을 했나
- **GPT-5.4 교사 연동**: `.env.azure`(gitignore) + `gen_llm`/`distill`/`aug_s` gpt-5 분기. 연결·라벨 품질 확인.
- **균형 증강**: kd2(S만 증강 → C재현율 하락) 교훈 반영. **S 250 + C경계 200 + O 50 = 500건** 신규 생성
  (gpt-4o 생성 + **GPT-5.4 라벨**). 교사셋 3,968 → 4,468건, S 366→599·C경계 강화.
- **재증류(kd3)**: 균형샘플+온도1.5+maxlen128, best-on-heldout 저장.

## 2. 결과 (held-out 72, 승격판 base=kd3)

| 관점 | F1 | C재현율 | 유출 | 비고 |
|---|---|---|---|---|
| 뉴럴-단독(maxlen 128, 학습 내부) | 0.839 | 0.958 | — | 128 truncation로 1건 놓침 |
| **뉴럴-단독(maxlen 256)** | **0.888** | **1.00** | 0 | 긴 문맥에선 뉴럴도 zero-leak |
| **전체 파이프라인(튜닝 앙상블)** | **0.873** | **1.00** | **0** | 프로덕션 기준(maxlen 512) |

- 프로덕션 `neural_infer`는 maxlen 512(전체 문서) → 긴 문맥 이점 그대로.
- **뉴럴 자체도 zero-leak**(256+) + 규칙 floor 이중 방어 → 옵션1 목표(0.8+ & 누락0) **달성**.

## 4. 무엇이 통했나 (교훈)
1. **C경계 문서 증강**: S만 늘리면 C/S 경계가 밀려 C재현율↓(kd2). **C경계를 함께** 늘리니 C재현율 회복.
2. **GPT-5.4 교사**: 더 강한 교사의 soft-라벨 → 경계 뉘앙스 전이 품질↑.
3. **튜닝 앙상블**(soft·뉴럴4.0): 뉴럴 실력을 전체 파이프라인에 반영(과대분류 억제).
4. **긴 문맥(maxlen↑)**: 문서 뒷부분 식별자 보존 → C재현율·정확도 동반 상승.
