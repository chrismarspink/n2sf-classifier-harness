## 6. 안전성 (불변 원칙)

- **기밀 재현율 1.00 · 과소분류(유출) 0** — 전 모델·전 언어.
- 뉴럴이 놓칠 경계 문서는 **규칙 floor**(주민번호·카드·기밀표지 등 언어무관 강식별자)가 이중 차단.
- 과대분류(공개→민감)는 사람 재검토로 흡수, 유출은 원천 차단이 원칙.

---

## 7. 방법 · 배포

**어떻게 만들었나**
- 클라우드 LLM(교사)이 O/S/C **확률분포** 부여 → 로컬 모델이 모방(soft-label 지식증류).
- 교사엔 **합성 문서만** 전송(실기밀 미사용) · 추론엔 LLM 호출 0.
- 클래스 균형 + C경계 + **다국어(영·중·일) 교사 라벨** 증강 → 누락 0 유지·다국어 발현.
- 3-tier(정규식→NER→증류 뉴럴) + 튜닝 앙상블(뉴럴 가중). 설명가능성=검출별 가법 기여도(SHAP).

**배포(온프레미스)**
- Hugging Face(공개/사설) `from_pretrained()` = docker pull 개념 / Docker 이미지 `docker run` 한 줄 / 에어갭 tar·오프라인 이미지.
- **GPU 불필요**, 일반 서버·PC CPU. 고객 정책은 GradeProfile로 주입.

## 8. 측정 주의 (정직성)

- held-out **72문서** 기준. 다국어(L5) 매우 높은 값(0.99~1.0)은 **다국어 학습 분포와 평가셋이 유사**해
  일부 in-distribution 성격 → 더 독립적인 **한국어 held-out(0.90~0.97)** 및 **도입처 실문서 검증**을 상용화 전 권장.
- xlmr-large 다국어 재증류(xlmr-ml)는 마무리 단계 — 완료 시 갱신.
